# Tag_HTML 
```html
Terdapat dua tag untuk memformat suatu teks, yaitu tag pembuka <tagname> dan tag penutup </tagname> .
Struktur dasar HTML:

<!DOCTYPE html> => Menandakan bahwa dokumen menggunakan HTML5.
<html> Dokumen HTML </html> => Tag utama sebagai pembungkus seluruh isi HTML.
<head> Header Dokumen </head> => Berisi informasi seperti title, tidak ditampilkan langsung di halaman.
<body> Isi Halaman </body> => Berisi semua konten yang ditampilkan pada halaman website.

Tag-tag pelengkap yang ada di HTML beserta fungsinya:

<title> untuk judul dari dokumen HTML </title> => Judul yang muncul di tab browser, biasanya ditaruh di bagian head.
<h1> Header Halaman atau judul </h1> => Digunakan untuk judul, tersedia dari <h1> sampai <h6> dengan ukuran berbeda (semakin besar angkanya, maka semakin kecil ukurannya).
<p> Paragraf </p> => Digunakan untuk membuat paragraf baru.
<b> Bold </b> => Menebalkan huruf/kalimat.
<i> Italic </i> => Memiringkan huruf/kalimat.
<u> Underline </u> => Menggaris bawahi huruf/kalimat.
<br> => Untuk membuat baris baru, tidak memerlukan tag penutup.
<ul> Unordered List </ul> => Digunakan untuk membuat daftar tidak berurutan (bullet).
<li> List Item </li> => Isi dari daftar pada <ul> atau <ol>.
<a href="..."> Link </a> => Digunakan untuk membuat hyperlink ke website atau email.
<img src="..." alt="..."> => Menampilkan gambar, tidak memiliki tag penutup.
<details> Detail </details> => Membuat konten yang bisa dibuka dan ditutup.
<summary> Ringkasan </summary> => Judul yang diklik untuk membuka isi <details>.
<footer> Bagian Bawah </footer> => Digunakan untuk bagian bawah halaman (misalnya tanggal, copyright).

```
