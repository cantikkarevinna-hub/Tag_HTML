# Tag_HTML 
```html
Terdapat dua tag untuk memformat suatu teks, yaitu tag pembuka <tagname> dan tag penutup </tagname> .
untuk dasar dan ketentuan keharusan dalam HTML yag wajib ada adalah tag html,head,body.

Tag-tag pelengkap yang ada di HTML beserta fungsinya:
<title> untuk judul dari dokumen HTML </title> =>  Judul yang muncul di tab browser, biasanya ditaruh dibagian header.
<h1> Header Halaman atau judul </h1> => bisa dibagian header, tetapi lebih sering di taruh dibagian body. untuk tag ini juga ada bermacam-macam ukuran 1(>) - 5(<).
<p> Paaragraf </p> => Digunakan untuk membuat paragraf baru.
<b> Bold </b> => Menebalkan huruf/kalimat.
<i> Italic </i> => Memiringkan huruf/kalimat.
<u> Underline </u> => Menggaris bawahi huruf/kalimat.
<br> => untuk garis kosong/baru, tidak diperlukan tag penutup.
```
